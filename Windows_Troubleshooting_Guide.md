# Windows Troubleshooting Guide

## 1. Basic Checks
- Restart the device  
- Verify power and cable connections  
- Confirm user is entering correct credentials  
- Check for error messages  

## 2. Network Issues
- Check Wi-Fi/Ethernet connection  
- Run `ipconfig /all`  
- Run `ping 8.8.8.8`  
- Restart router or adapter  
- Renew IP: `ipconfig /renew`  

## 3. Performance Issues
- Check Task Manager for high CPU/RAM  
- Close unnecessary applications  
- Clear temp files: `temp`, `%temp%`  
- Run Windows Troubleshooter  

## 4. Application Issues
- Restart the application  
- Check for updates  
- Reinstall if needed  
- Review Event Viewer logs  

## 5. Login Issues
- Verify CAPS LOCK  
- Reset password  
- Check domain connection  
- Unlock account in Active Directory
