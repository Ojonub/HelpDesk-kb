# Active Directory User Management

## 1. Reset Password
- Open AD Users & Computers  
- Locate user  
- Right-click → Reset Password  
- Ensure "User must change password at next logon" is selected  

## 2. Unlock Account
- Right-click user → Properties  
- Check "Account is locked out"  
- Unlock  

## 3. Create New User
- Right-click OU → New → User  
- Enter details  
- Assign temporary password  
- Add to appropriate groups  

## 4. Add User to Group
- Right-click user → Add to a group  
- Search group name  
- Confirm membership  

## 5. Disable Account
- Right-click user → Disable Account
