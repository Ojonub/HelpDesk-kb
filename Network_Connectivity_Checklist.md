# Network Connectivity Troubleshooting Checklist

## 1. Physical Checks
- Ensure cables are connected  
- Check router/modem lights  
- Verify Wi-Fi is enabled  

## 2. Software Checks
- Check network icon status  
- Disable/enable adapter  
- Forget and reconnect to Wi-Fi  

## 3. Command-Line Tools
- `ipconfig /all`  
- `ping 8.8.8.8`  
- `ping <gateway>`  
- `ipconfig /release` + `ipconfig /renew`  

## 4. Advanced Checks
- Check firewall settings  
- Test with another device  
- Restart router or PC
