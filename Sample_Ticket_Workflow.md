# Sample Ticket Workflow

## 1. Ticket Received
- User submits issue via phone, email, or portal  

## 2. Categorize & Prioritize
- Determine severity  
- Assign appropriate category  

## 3. Troubleshoot
- Follow KB articles  
- Document steps taken  

## 4. Resolve or Escalate
- Resolve if within scope  
- Escalate to Tier 2 if needed  

## 5. Close Ticket
- Add resolution notes  
- Confirm with user
