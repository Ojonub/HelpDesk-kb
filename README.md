# Help Desk Troubleshooting Knowledge Base (KB)

This project is a collection of practical troubleshooting guides and workflows I created after completing the **Redwood IT Help Desk Technician Training** and the **TCM Practical Help Desk Training**. It demonstrates my understanding of real-world Help Desk processes, documentation, and structured problem-solving.

The goal of this repository is to showcase:
- My understanding of Windows, networking, and Active Directory fundamentals  
- My ability to communicate technical steps clearly and professionally  
- My ability to troubleshoot common IT issues  
- My familiarity with ITSM-style documentation  

---

## Repository Contents

### 1. Windows_Troubleshooting_Guide.md  
Step-by-step instructions for resolving common Windows issues such as login failures, slow performance, connectivity problems, and application errors.

### 2. Active_Directory_User_Management.md  
Basic user-management tasks including password resets, account unlocks, group membership updates, and account creation scenarios.

### 3. Network_Connectivity_Checklist.md  
A structured checklist for diagnosing network issues using physical checks, command-line tools, and basic network concepts.

### 4. Sample_Ticket_Workflow.md  
A simple ITSM-style workflow showing how incidents are received, categorized, documented, resolved, or escalated.

### 5. Remote_Support_Best_Practices.md  
Guidelines for assisting users remotely, maintaining professionalism, and ensuring secure troubleshooting.
